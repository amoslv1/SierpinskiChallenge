## Carpet

### Size
* Smallest Unit Size ^ Order = Carpet Size
* Size must be a square/cube etc number

| Smallest Unit Size | Order | Carpet Size     |
|--------------------|-------|-----------------| 
| 3                  | 2     | 9               |
| 3                  | 3     | 27              |
| 3                  | 4     | 108             |
| 3                  | 5     | 324             |
